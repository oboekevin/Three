OcuPhysics

Summary

The goal of this project was to model various electrically-charged objects and the electric field they generate in 3D using the Oculus Rift. The idea was born out of a difficulty understanding traditional 2D diagrams. The project uses Three.js for the graphics and connection to the Oculus. It only models point charges, but there is a smaller side application that show the electric field around a charged solenoid.

Camera features

  • Zoom in
  • Zoom out
  • Pan
  • Turn
  • Select & change
